🍕 **Pizza Sales Analysis (SQL)**

📌 **Project Overview**

This project focuses on end-to-end analysis of pizza sales data using SQL. The objective is to extract meaningful business insights related to sales performance, customer preferences, and ordering behavior.

Using SQL operations like JOINs, GROUP BY, and aggregate functions, raw data is transformed into actionable insights to support data-driven decision-making.

🎯**Objectives**

- Analyze overall sales performance
  
- Understand customer ordering pattern
  
- Identify top-performing pizzas and categories

- Discover trends based on time and demand

🛠️ **Tools & Technologies**

SQL

Relational Database

Data Analysis Techniques

📊 **Key Analysis Performed**

🔹 **Basic Analysis**

- Total number of orders placed

- Total revenue generated
  
- Highest-priced pizza
  
- Most common pizza size ordered
  
- Top 5 most ordered pizza types

🔹 **Intermediate Analysis**

- Total quantity of each pizza category ordered
  
- Distribution of orders by hour
  
- Category-wise pizza distribution
  
- Average pizzas ordered per day
  
- Top 3 pizzas based on revenue
  
🧠 **Key SQL Concepts Used**

- JOINs – to combine multiple tables
  
- GROUP BY – for categorization and summarization
  
- Aggregate Functions – SUM(), COUNT(), AVG()
  
- Filtering & Sorting – WHERE, ORDER BY
  
- Subqueries – for advanced insights
  
📈 **Insights Gained**

- Identified peak ordering hours
  
- Found most popular pizza sizes and categories
  
- Determined top revenue-generating pizzas
  
- Analyzed daily and hourly sales trends

🚀 **Conclusion**

This project demonstrates how SQL can be effectively used to analyze structured data and derive valuable business insights. It highlights the importance of data-driven strategies in improving sales and customer satisfaction.
